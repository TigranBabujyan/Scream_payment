# Scream Payment

**React payment-flow prototype built as a portfolio project for a canyoning / adventure booking experience.**

Scream Payment is an older frontend project focused on checkout flow, form handling, responsive UI, Firebase integration, and multilingual interface work.

---

## Project Status

This project is not a finished production application. It is kept public as a portfolio prototype that shows my early React workflow, feature iteration, and experience building real user flows.

The codebase is useful as a snapshot of my earlier frontend development stage before moving into newer Next.js and AI-focused projects.

---

## What It Demonstrates

- React application structure with Create React App
- TypeScript-based frontend development
- Payment / booking flow UI
- Form-driven user interactions
- Firebase integration
- Internationalization with i18n / translation libraries
- Responsive UI work
- Iterative development through branches and pull requests

| Layer | Technology |

## Getting Started

### Prerequisites

- Node.js 16+
- npm

### Installation

```bash
git clone https://github.com/TigranBabujyan/Scream_payment.git
cd Scream_payment
npm install
```

### Run locally

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000).

---

## Available Scripts

```bash
npm start      # Start development server
npm run build  # Build production bundle
npm test       # Run test runner
```
